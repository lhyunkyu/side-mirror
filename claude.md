# 화면 프라이버시 보호 기능 요구사항 명세

## 1. 경고 모드 (Warning Mode)
다음 조건을 만족하면 즉시 화면에 경고를 표시합니다.

*   **진입 조건:** 총 감지 인원 >= 2명 상태가 **3초 이상 연속** 유지될 때
*   **동작:**
    *   지연 시간이나 등장 애니메이션(Fade In 등) 없이 즉시 경고 UI 표시
    *   침입자가 감지된 방향에 맞추어 경고 표시

## 2. 개인정보 보호 모드 (Privacy Mode)
경고가 무시되고 침입 상황이 지속될 경우, 화면을 강제로 숨깁니다.

*   **진입 조건:** Warning 모드 진입(경고 표시) 이후에도 **2명 이상 감지 상태가 2초 이상 더 유지**될 때 (총 5초 이상 연속 감지)
*   **동작:**
    *   민감한 화면 노출 방지를 위해, 현재 사용 중인 앱에서 자동으로 벗어나 **macOS 바탕화면(Desktop)으로 강제 전환(Show Desktop)** 합니다.

## 3. 안전 상태 복귀 (Safe Mode)
침입자가 사라져 다시 안전해지면 경고를 해제합니다.

*   **진입 조건:** 화면에 사용자만 존재하거나 아무도 감지되지 않을 때
*   **동작:**
    *   경고 UI를 부드럽게 제거합니다.
    *   **애니메이션 효과:** Opacity 100% → 0%, Duration 약 0.3~0.5초, Ease Out
    *   *(참고: 바탕화면으로 전환된 상태인 경우, 이전 앱으로 자동 복귀하지 않고 경고 UI만 Fade Out 처리합니다.)*

---

## 4. 전체 상태 전이 흐름 (State Flow)

[ Safe ]
  ↓ (2명 이상 감지 후 3초 연속 유지)
[ Warning ] 
  - 경고 UI 즉시 표시 (등장 애니메이션 없음)
  ↓ (2명 이상 감지 상태 2초 추가 유지)
[ Privacy Mode ] 
  - macOS 바탕화면(Desktop)으로 자동 전환
  ↓ (뒤에 있던 사람이 사라짐)
[ Warning Fade Out ]
  - 0.3~0.5초, Ease Out으로 경고 UI 소멸
  ↓ 
[ Safe ]

# 5. 메뉴 막대 (Menu Bar) 제어

macOS 상단 메뉴 막대(상태 표시줄)에 앱 아이콘을 표시하고, 클릭 시 드롭다운으로 다음 제어 메뉴를 제공합니다.

*   **일시정지 (Pause)**
    *   카메라 및 사람 감지 로직을 일시적으로 중단합니다.
    *   UI 피드백: 메뉴 막대의 앱 아이콘을 회색으로 흐리게 처리하거나 다른 아이콘으로 변경하여, 사용자가 현재 감지가 중지되었음을 직관적으로 알 수 있게 합니다.
*   **다시시작 (Resume)**
    *   일시정지된 감지 기능을 다시 활성화하고 초기 상태(Safe)로 복구합니다.
    *   UI 피드백: 메뉴 막대의 앱 아이콘을 원래 활성화 상태로 되돌립니다.
*   **종료 (Quit)**
    *   앱을 완전히 종료하고, 사용 중이던 카메라 리소스와 백그라운드 프로세스를 안전하게 해제합니다.

> **구현 참고 (UI/UX)**
> '일시정지' 상태일 때는 메뉴에 '다시시작'만 보이고, 정상 작동 중일 때는 '일시정지'만 보이도록 메뉴 항목을 토글(Toggle) 방식으로 처리하는 것을 권장합니다.

## 1. 경고 모드 (Warning Mode)
다음 조건을 만족하면 즉시 화면에 방향성 경고 UI를 표시합니다.

*   **진입 조건:** 총 감지 인원 >= 2명 상태가 **3초 이상 연속** 유지될 때
*   **동작 및 UI 요구사항 (자동차 사이드미러 스타일):**
    *   지연 시간이나 등장 애니메이션(Fade In 등) 없이 즉시 경고 UI를 표시합니다.
    *   **방향성 시각화:** 사람이 감지된 위치에 따라 **화면 왼쪽(Left Edge) 또는 오른쪽(Right Edge) 가장자리**에 반투명한 경고 바(Bar)나 인디케이터를 띄웁니다.
    *   (예: 자동차 후측방 경고등처럼, 오른쪽에 사람이 감지되면 모니터 오른쪽 끝에 붉은색/황색 계열의 경고 그래픽이 깜빡이거나 켜짐)
